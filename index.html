Device

Device detection plugin. The device detection is done according to the window's width, no UA parsing is used.

Dependencies

  • Browser.js
  • underscore - http://underscorejs.org/
  • jQuery - http://jquery.com/

Methods

device_type():String
Gives current Device
is((string) device_type):Boolean
Checks if the current device equals device_type. device_type is a String representing the device you want to test, eg. "mobile"
refresh():void (private)
Forces Device to recalculate the device_type.

Properties

is_DEVICE_TYPE (boolean)
For each device type Device creates properties with the following syntax "is_DEVICE_TYPE", where DEVICE_TYPE is a kind of device. e.g Device.is_mobile

Demo

The following box's text will change when a Device change occurs


HTML

<div class="demo_box">
</div>

JS

$(".demo_box").html(Device.device_type());
$(window).on("device_change",function(event){
  $(".demo_box").html(event.device);
})
